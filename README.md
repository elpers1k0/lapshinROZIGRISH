# Система для розыгрыша призов

Курсовой проект для Хекслет колледжа, ИСИП 01.24, разработал Лапшин Кирилл

Приложение для случайного выбора победителя из списка участников.

## Как запустить

1. Убедитесь, что у вас установлен Node.js.
2. Откройте консоль (терминал) в папке проекта.
3. Запустите команду:
   ```bash
   node index.js
   ```

## Файлы с данными

- `participants.json` - список участников в формате JSON.
- `history.json` - сюда автоматически сохраняется история розыгрышей.

## Команды меню

- **1** - загрузка участников из JSON и запуск розыгрыша.
- **2** - вывод истории розыгрышей.
- **3** - выход из приложения.
